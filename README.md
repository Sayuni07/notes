## Keyboard shortcuts 
Shortcuts I frequently use:
- Ctrl+C (copy)
- Ctrl+V (paste)
- Ctrl+T (new tab)
- Ctrl+N (new window)
- Ctrl+A (select all items)
- Ctrl+B (bold text)
- Ctrl+U (underline text)

Shortcuts I would like to start using:
- Ctrl+E (open search)
- Ctrl+Del (delete words to the right)
- Ctrl+H (find and replace text)
